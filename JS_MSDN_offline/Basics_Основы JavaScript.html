    Основы JavaScript

     

    JavaScript — это интерпретируемый объектный скриптовый язык.

    В языке JavaScript используется синтаксис, аналогичный синтаксису языка C; он поддерживает структурированные конструкции, такие как if...else, for и do...while. Фигурные скобки ({}) используются для ограничения блоков операторов. Язык поддерживает различные типы данных, включая String, Number, Boolean, Object и Array. Также поддерживаются расширенные функции работы с датами, тригонометрические функции и регулярные выражения.

    Вместо классов в JavaScript используются прототипы. Определить объект можно путем создания функции конструктора.

    JavaScript является слабо типизированным языком, что означает, что типы данных переменных не объявляются явно. Во многих случаях JavaScript выполняет преобразования автоматически, когда они необходимы. Например, при добавлении числа к элементу, состоящему из текста (строка), число преобразуется в текст.
